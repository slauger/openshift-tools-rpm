OpenShift Tools RPM Repository

Pre-built RPM packages for common OpenShift and Kubernetes CLI tools.

Quick Setup

sudo dnf config-manager --add-repo https://slauger.github.io/openshift-tools-rpm/openshift-tools.repo

Or manually create /etc/yum.repos.d/openshift-tools.repo:

[openshift-tools]
name=OpenShift Tools
baseurl=https://slauger.github.io/openshift-tools-rpm/
enabled=1
gpgcheck=0

Install packages

# Install individual tools
sudo dnf install helm kubectl argocd-cli k9s

# Install all available tools
sudo dnf install helm openshift-clients kubectl yq kubectx argocd-cli \
  tekton-cli virtctl trivy conftest velero stern k9s kustomize \
  knative-client cosign kubeseal
